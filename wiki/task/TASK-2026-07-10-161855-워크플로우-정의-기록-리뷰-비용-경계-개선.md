## 개요

기존 플러그인을 물리적으로 분리하지 않고 task-github define을 DefinitionArtifact 생성 단계로 바꾸며 local/github 소비 경로를 제공한다. session-review에는 리뷰 에피소드 단위 reviewer lease를 추가하고 각 워크플로우 경계에 추정 없는 비용 receipt를 연결한다.

## 근거

사용자는 결과 품질을 최우선으로 두되 토큰, 시간, 개입 비용을 줄이길 원한다. GitHub 기록이 필요하면 root 전체 tree가 완전해야 하고, 필요 없으면 동일 define 결과를 로컬에서 소비해야 한다.

## 범위와 완료 기준

완료 기준: DefinitionArtifact의 안정 ID/digest/dependency 검증, local ready/start/complete 상태 전이, GitHub full-tree coverage/resume gate, reviewer lease reuse/invalidation 검증, unknown token=null 비용 receipt, legacy 회귀 테스트와 배포 메타데이터 정합성. 영향 경로: plugins/task-github/**, plugins/session-review/**, plugins/studio/**, marketplace manifests.
